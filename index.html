AJAX

Ajax call codeigniter function echo html.

            
                $.ajax({
                  url: "<?php echo base_url() . $this->modul_name . '/getDetAngsuran'; ?>",
                  type: 'POST', //method: 'POST',
                  data: {
                  angsuran_id: angsuran_id
                  },
                  success: function(res) {
                    alert(res);
                  }
                });

Ajax call codeigniter function json.

            
                $.ajax({
                  url: "<?php echo base_url() . $this->modul_name . '/getDetAngsuran'; ?>",
                  type: 'POST',
                  dataType: "json",
                  data: {
                    angsuran_id: angsuran_id
                  },
                  success: function(res) {
                    alert(res.angsuran_id);
                    $('#id_angsuran').html(res);
                  }
                });

Background Video HTML 5

Membuat background video dengan HTML 5 agar terlihat lebih dinamis
Contoh web: http://demosthenes.info/samples/polina.html
Menggunakan tag <video></video>
Contoh script:
<video autoplay loop poster=”/assets/images/polina.jpg id=”bgvid”>
<source src=”../assets/videos/polina.webm” type=”video/webm”>
<source src=”../assets/videos/polina.mp4″ type=”video/mp4″> </video>
