Setting PHP .ini

Cari file php.ini

Cari:
post_max_size = 8M
upload_max_filesize = 2M
max_execution_time = 30
max_input_time = 60
memory_limit = 8M


Ubah nilai menjadi:
post_max_size = 750M
upload_max_filesize = 750M
max_execution_time = 5000
max_input_time = 5000
memory_limit = 1000M

short_open_tag=Off

short_open_tag=On
